# Raccoon

This tool is an advanced packet sniffer with a GUI interface for capturing and analyzing network traffic in real-time.
## Installation



```bash
  git clone https://github.com/sanskar-sudo/Raccoon.git
  cd Raccoon
  pip3 install -r requirements.txt
  python3 Raccoon.py
```
